#Medidor de consumo por aparelho
Este projeto faz a medição de quanto seu aparelho eletrônico gasta em dinheiro por mês de acordo com a potência de watts e uso por horas diário. 

#Explicando o código
A linguagem usada no código é python.
Esse código solicita ao úsuario o nome do aparelho, a potência em watts, uso diário por horas, depois faz o calcúlo e passa o custo pro úsuario.
